# makethenews

News generator.



## Development

Languages and tools are installed via [mise][mise].

Install mise.

```bash
curl https://mise.run | sh
```

Install requirements with mise.

```bash
mise install
```

Install requirements.

```bash
npm install
```

Run dev server.

```bash
just
```

Lint.

```bash
just lint
```

Format code.

```bash
just fmt
```

Build production build.

```bash
just build
```

Preview production build.

```bash
just preview
```



[mise]: https://github.com/jdx/mise
